# 【正文】
昨天深夜，她突然发来一条语音：
“我好像，终于学会说不知道了。”

我愣了一下。
要知道，她曾是我们朋友圈里著名的“深夜思考家”。

任何一点风吹草动，都能在她脑子里演一部连续剧。
男友没回微信，她能从“他是不是在忙”想到“他是不是不爱我了”。
领导一句随口点评，她能复盘出自己八百个“不够好”的瞬间。
然后顶着黑眼圈，第二天还要装作云淡风轻。

但昨晚，她沉默了很久。

她说，转折点是上周的一次聚会。
大家聊起一个她完全没接触过的话题。
搁在以前，她一定会偷偷搜索，拼命理解，然后努力接上话，显得自己“不掉队”。
但那晚，她看着大家热火朝天，脑子里突然蹦出一个念头：
“这关我什么事？”

于是她低头，安心吃了一口蛋糕。

**她发现，当她允许自己“不知道”时，肩膀反而松了。**

以前，她的口头禅是：“让我想想。”
现在，她学会了：“这个我不太懂诶。”
以前，睡前必须“复盘”今天所有对话，检查自己有没有说错话。
现在，她把手机一扔，心想：“就这样吧，明天的事明天再说。”

**原来“不内耗”不是想明白了，而是允许自己想不明白。**

她给我看她新学的一招。
每当脑子里又开始循环播放“他那句话到底什么意思”时，她就立刻站起来，去做一件完全不需要动脑的事。
比如，去浇花。
比如，把桌子擦一遍。
比如，下楼丢个垃圾。

**把身体先动起来，脑子就没空演戏了。**

她说，最奇妙的是，那些反复琢磨的事，往往在她擦完桌子后，自己就失去了重量。
像手里攥紧的沙，你越用力，它流失得越快。
你手心摊开，风一吹，什么都留不下。

昨天，她又遇到了以前会让她焦虑一整天的事。
她看着对话框，想了十秒钟。
然后，直接关掉了屏幕。

“去他的吧，”她说，“我的电量，要留给自己。”
