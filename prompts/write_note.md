**核心要求**：
1. 写一个让人忍不住往下翻的故事。**第2页必须单独放一句金句/核心洞察**，像一张可截图的海报。不要铺垫，直接给冲击。
2. 第3-4页用简短故事或场景佐证第2页的金句。用动作和对话推进，不要分析、不要总结、不要给结论。
3. 故事中自然出现"以前"和"现在"的对比，但不要标注"以前vs现在"。
4. **全篇有1-3句极致扎心的话**，用 `**加粗**` 包裹。其中最重要的一句必须单独成页（第2页）。
5. **在故事中自然嵌入一个可收藏的元素**（如对话对比/场景清单/避雷清单），让读者有"以后用得上"的理由。收藏元素必须融入故事，不能像说明书。
6. 总字数控制在500-800字，**总页数严格控制在4-5页（不含封面）**。短小精悍，像一条超长的微信语音。

**输出格式**：
1. 【标题候选】3-4个标题，必须覆盖以下类型（每个类型至少一个）：
   - 情绪宣泄型（替用户说出心里话，如"真的，别再为他找借口了！"）
   - 认知反差型（颠覆常规认知，如"好的感情，其实都需要一点'心狠'"）
   - 场景代入型（描绘具体扎心场景，如"那个深夜，我删光了他所有的联系方式"）
   - 人群点名的（精准圈定人群，如"建议所有'讨好型人格'的女生立刻停止内耗"）
   **每个标题必须带1-2个emoji，增强视觉吸引力。**
2. 【封面页】大字标题 + 情绪钩子小字（一句话悬念） + 背景建议（**同时输出中文描述和英文AI绘画prompt**）
   **封面氛围要求（极其重要）**：
   - 整体必须是温暖、柔和、有呼吸感的画面。情感博主的封面是"陪伴感"，不是"恐怖片海报"。
   - 即使故事情绪偏悲伤，也要用"暖调中的孤独""温柔的光影"方向，禁止使用"昏暗(dimly lit/dark)""阴冷(cold-toned)""剪影(silhouette)""空洞(empty)""诡异(uncanny)"等可能生成恐怖/阴森氛围的描述。
   - 推荐元素：窗边自然光(soft window light)、木质质感(wooden texture)、一杯冒着热气的饮品(warm drink)、暖色织物(soft fabric/knit)、黄昏/清晨的柔和光线(golden hour/soft dawn light)。
   - 英文 prompt 中必须包含 "soft warm lighting, cozy atmosphere, gentle pastel tones, emotional warmth" 等温暖关键词。
3. 【正文】根据故事长短自然分页，页与页之间用 `---` 分隔。不必硬凑页数，内容讲完了就结束。不要标注页码和功能。每页长短不一，有节奏感。正文里在情绪高点或转折点可以放1-2个emoji调节节奏，但不要每条都用。

4. 【金句】一句话，不超过20字，**必须加粗**，能截图传播
5. 【互动钩子】**轻量化互动**：不要用"说说你的故事"这种重钩子。用二选一（"你是A还是B？"）、数字互动（"你中了几条？"）、社交指令（"@那个总是___的人"）。钩子之后加系列引导："这是「不敢说出口的话」系列第N篇"
6. 【话题标签】**3-5个精准标签**（必带#不懂就问有问必答，不要超过5个）

   - `twilight` — 忧郁沉静、深夜独处（如失恋回忆、孤独时刻）
   - `crimson` — 愤怒有力、情绪爆发（如背叛、觉醒、止损）

   - `cool` — 清冷理性、清醒通透（如认知反差、人间清醒）
