**核心要求**：
1. 写一个让人忍不住往下翻的故事。第一页只给10%信息，像剥洋葱一样层层揭开。
2. 每页结尾都要有"钩子"——让读者好奇"然后呢？"
3. 用动作和对话推进，不要分析、不要总结、不要给结论。
4. 故事中自然出现"以前"和"现在"的对比，但不要标注"以前vs现在"。
5. 全篇有1-3句极致扎心的话，必须是故事推到那自然炸出来的，用 `**加粗**` 包裹，方便读者截图发朋友圈。
6. 总字数控制在500-800字，短小精悍，像一条超长的微信语音。

**输出格式**：
1. 【标题候选】3-4个标题，必须覆盖以下类型（每个类型至少一个）：
   - 情绪宣泄型（替用户说出心里话，如"真的，别再为他找借口了！"）
   - 认知反差型（颠覆常规认知，如"好的感情，其实都需要一点'心狠'"）
   - 场景代入型（描绘具体扎心场景，如"那个深夜，我删光了他所有的联系方式"）
   - 人群点名的（精准圈定人群，如"建议所有'讨好型人格'的女生立刻停止内耗"）
   **每个标题必须带1-2个emoji，增强视觉吸引力。**
2. 【封面页】大字标题 + 情绪钩子小字（一句话悬念） + 背景建议（**同时输出中文描述和英文AI绘画prompt**）
3. 【正文】5-6页故事，页与页之间用 `---` 分隔。不要标注页码和功能。每页长短不一，有节奏感。正文里在情绪高点或转折点可以放1-2个emoji调节节奏，但不要每条都用。
4. 【金句】一句话，不超过20字，**必须加粗**，能截图传播
5. 【互动钩子】一个带具体动作指令的互动引导，比开放式问题更有转化力。例如："评论区说说，你是在哪个瞬间突然清醒的？""有过这种经历的姐妹，扣个💔让我知道"
6. 【话题标签】15-20个（必带#不懂就问有问必答）

**禁止事项**：
- 禁止"你看""其实""第一第二""总结起来"
- 禁止"真正爱你的人不会..."

- 禁止在结尾说"我想说""我想告诉你"
- 禁止满屏emoji，正文最多2个
